# AgriFarm e-Connect
## Languages Used
- HTML
- CSS/BOOTSTRAP other Frameworks
- Javascript
- PHP
- MySQL
## Features
### User-Side
#### 
- Login
- Register
- Forgot Password
- Purchase Product
- Add to Cart
- Customer Support
- Wishlist (Save Products)
- Notification From Admin
- Logout

### Super-Admin Side
- All admin Previleges
- Can remove Users
- Can add Sub Admin

### Sub-Admin Side
- Can Add Products
- Can Update Products
- Can View the Users Message
- Can Update the Pending orders
